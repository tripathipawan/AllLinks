# 🔗 Tripathi Dev Lab — AllLinks

A clean, minimal, and fully responsive link-in-bio page for **Tripathi Dev Lab** — a developer content brand. Built with pure HTML, CSS, and JavaScript, this single-page site centralizes all social media and professional profile links in one shareable URL.

**🔗 Live Demo:** [tripathidev-lab.vercel.app](https://tripathidev-lab.vercel.app/)

---

## 📌 Overview

AllLinks is a custom-built alternative to tools like Linktree, designed specifically for the Tripathi Dev Lab brand. It presents a branded, visually cohesive landing page with the developer's profile, tagline, and direct links to all active platforms — Instagram, WhatsApp Channel, YouTube, Facebook, GitHub, LinkedIn, and CodePen.

The page is intentionally lightweight with zero dependencies — no npm, no frameworks, no build step. Open `index.html` and it works.

---

## ✨ Features

- **Branded Profile Header** — Displays the Tripathi Dev Lab logo, name, and brand tagline ("Code You Can See") at the top of the page for immediate visual identity.
- **7 Platform Links** — Direct, clickable links to all active social and professional profiles: Instagram, WhatsApp Channel, YouTube, Facebook, GitHub, LinkedIn, and CodePen — all opening in a new tab.
- **Hover Animations** — Each link card has smooth CSS hover transitions, giving the page an interactive and polished feel without JavaScript.
- **Fully Responsive Layout** — CSS media queries ensure the layout renders correctly on mobile, tablet, and desktop screens. Optimized for mobile-first usage since link-in-bio pages are primarily accessed via phones.
- **Custom Styling** — The visual design is built from scratch with a custom `Style.css`, including typography choices, button styling, spacing, and brand-consistent color usage.
- **JavaScript Interactions** — `Script.js` handles any dynamic behaviors on the page, such as link interaction effects or UI state changes.
- **Static Assets** — All images and icons are stored in the `assets/` folder, keeping the project self-contained and portable.
- **Vercel Deployed** — The site is deployed to Vercel for fast global CDN delivery and instant load times.
- **Zero Dependencies** — No npm packages, no build tools, no external libraries required. The entire project runs by opening `index.html` in any browser.
- **Shareable Single URL** — The entire page is designed to be shared as a single link — ideal for social media bios, email signatures, and business cards.

## 🚀 Getting Started

No installation or build step required.

**1. Clone the repository**
```bash
git clone https://github.com/tripathipawan/AllLinks.git
```

**2. Navigate into the project folder**
```bash
cd AllLinks
```

**3. Open the page**

Simply open `index.html` in any modern browser — or use the VS Code **Live Server** extension for a better development experience:

```
Right-click on index.html → Open with Live Server
```
